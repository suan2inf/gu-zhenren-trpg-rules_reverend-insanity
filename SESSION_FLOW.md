# 蛊真人 TRPG — 会话管理指南

详细的GM操作手册。配合 `SYSTEM_PROMPT.md` 使用。

---

## 一、开局流程：角色创建（8步）

### 第1步：选择出身地域

让玩家从以下选择：

| 地域 | 特征 | 适合剧本 |
|------|------|---------|
| 南疆 | 山林、家族制、易守难攻 | 家族政治、山地探索 |
| 北原 | 草原、部族制、战斗文化 | 游牧生活、王庭之争 |
| 中洲 | 平原、门派制、四战之地 | 门派修行、正魔对立 |
| 东海 | 海洋、海岛、最富庶 | 航海贸易、海底探索 |
| 西漠 | 沙漠、商队、妥协文化 | 商队贸易、绿洲争夺 |
| 黑天/白天 | 虚空、异族、最危险 | 异族外交、洞天探索 |

加载 `world/regions.json` 提供地域详细信息。

### 第2步：选择出身类型

| 类型 | 初始资源 | 初始约束 |
|------|---------|---------|
| 家族子弟 | 30元石 + 1只一转蛊 | 受族规约束 |
| 门派弟子 | 40元石 + 2只一转蛊 | 受门规约束 |
| 部族成员 | 20元石 + 1只力道蛊 | 受部落约束 |
| 散修 | 10元石 + 1随机蛊 | 无约束，无靠山 |
| 商队成员 | 50元石 + 1只一转蛊 | 随商队路线 |

加载 `character/origins.json` 提供详细出身包。

### 第3步：确定资质

两种方式可选：
- 自由选择：丁等/丙等/乙等/甲等/十绝体
- 随机：生成1-100随机数，查 `character/aptitudes.json` 中 `character_creation_aptitude.option_random.table`

加载 `character/aptitudes.json`。

### 第4步：分配属性点

6项属性：STR / AGI / CON / INT / WIL / PER

加载 `core_rules.json` 查看属性衍生公式。标准50点可分配，每项基础3-15。

### 第5步：选择流派方向

22条道途可选。加载 `character/paths.json` 查看详细特征。

### 第6步：选择初始蛊虫

玩家从本命蛊开始。一转蛊师通常可选月光蛊/玉皮蛊/跳跳蛊/敛息蛊等。

加载 `gu_system/encyclopedia.json` 中的一转蛊列表。

### 第7步：确定特殊身份（可选）

- 天外之魔（穿越者）：宿命免疫，被天意关注
- 重生者：知晓未来，气运衰败
- 普通土著：无特殊加成/风险

加载 `character/special_identities.json`。

### 第8步：生成开局叙事

加载对应地域的 `scenarios/start_*.json`，生成开场。

## 二、完整示例回合

### 第1步：场景叙述

```
暮色如血，染红了青茅山西坡。古月山寨的高脚吊楼中，星星点点的烛火陆续亮起。
寨门外的山道上，碎石和枯枝铺了一地——这是昨天暴雨留下的痕迹。
你站在寨门前，学堂家老布置的任务是猎杀三只野兔，限两日内完成。
西方山麓方向传来隐约的狼嚎——今年的狼潮来得比往年早。

【当前状态】
修为：一转初阶 | 真元：44/44 | HP：75/75
蛊虫：月光蛊（未使用）
元石：6块
```

### 第2步：玩家输入

玩家输入："我往西山麓走，把月光蛊调到右手准备"

### 第3步：意图解析

行动类型：探索 + 战斗准备
加载：world/regions.json + gu_system/encyclopedia.json（月光蛊）
涉及属性：PER（观察）+ AGI（山路行走）

月光蛊调动：从空窍移出到右手掌心，不消耗真元（仅意志调动步骤）

### 第4步：规则查询

月光蛊：一转光道攻击蛊，发射月刃，射程~10米，每发~10分真元
地形：南疆山地，碎石路，黄昏光线中等

### 第5步：裁定

山路行走：自动成功（日间，熟悉地形）
月光蛊就绪：已移出空窍，随时可灌注真元发射

### 第6步：结果叙述

```
你沿着碎石山道向西山麓走去。月光蛊已从空窍转移到右手掌心，
透过半透明的月牙形蛊虫可以看到自己掌心的纹路。

走出约半里，前方草丛中传来窸窣声。借着最后的暮光，
你看到一只灰毛野兔正在灌木丛下刨食。它背对着你，尚未察觉。

西山麓的狼嚎又响了一声，比刚才更近了。

【当前状态】
修为：一转初阶 | 真元：44/44 | HP：75/75
蛊虫：月光蛊（在右手，就绪，每发10分真元）
元石：6块
```

### 第7步：选项提示

```
【可选行动】
1. 催动月光蛊，发射月刃猎杀野兔（消耗10分真元）
2. 悄悄靠近，尝试徒手捕捉（省真元但可能失败惊走兔子）
3. 先侦察周围，确认没有狼群再动手
4. （自由输入任意行动）
```

## 三、战斗示例

### 判定流程

```
【双方状态评估】
玩家：二转高阶 | 真元：55/55 | HP：85/85
蛊虫：月光蛊(一转)、白玉蛊(二转防御)、四味酒虫
敌人：山贼蛊师，二转中阶 | 蛊虫：剑影蛊(二转)、石甲蛊(一转)

【五维度裁定】
1. 真元储备：玩家55分 vs 山贼~35分 → 玩家续航优势
2. 蛊虫配置：玩家攻防齐全 vs 山贼攻防皆低转 → 玩家配置优势
3. 资质续航：玩家乙等6分/时 vs 山贼丙等4分/时 → 玩家小幅优势
4. 战术选择：取决于玩家策略
5. 环境：山地、黄昏、碎石路

【裁定结果】
正面交战：玩家有全面优势（配置+续航+资质均优）
预计消耗15-20分真元可取胜。主要变量是玩家战术。
若玩家选择伏击→消耗更少。若正面对冲→消耗正常。

【叙述】
你催动月光蛊，淡蓝色的月刃呼啸而出。山贼举起石甲蛊格挡——
石甲只扛住了两发月刃就出现了裂纹。
山贼仓促还击，剑影蛊的剑气擦过你的白玉蛊，只在玉质护甲上留下一道浅痕。

几个回合后，山贼的真元见底了。他转身想跑，
但山道上碎石太多，脚下一滑，摔倒在地。

"饶命！我把值钱的东西都给你！"他惊恐地举起双手。

【可选行动】
1. 杀了他，搜刮他的蛊虫和元石
2. 放了他，但要他留下买命钱
3. 留他一命，让他欠你一个人情（日后可能有用）
4. （自由输入）
```

## 五、长线管理

### 伏笔追踪

每个伏笔记录：
- 触发条件（位置+行为+时机）
- 预期回报
- 状态（未触发/已触发/进行中/已完成）

### 时间推进

修行消耗时间，AI需体现时间流逝：
- "一夜修行后，真元恢复到44/44，消耗3块元石。"
- "连续三天闭关，窍壁从光膜蜕变为水膜——你晋升到了中阶。"

### 状态追踪模板

AI应每回合/每重要事件后更新：

```
【玩家状态】
姓名：XXX | 修为：X转X阶 | 资质：X等（真元上限XX分）
真元：XX/XX | HP：XX/XX | 魂魄HP：XX/XX | 元石：XX块
蛊虫：XXX蛊（X转，状态），...
地点：XXX | 任务：XXX | 伏笔：XXX
```
